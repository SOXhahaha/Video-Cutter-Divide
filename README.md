# Video Cutter Divide
视频片段分类工作台 — 批量浏览、切片、OCR 字幕识别、分类导出。

基于 Electron + TypeScript 构建，支持 GPU 加速 OCR 和 NVENC 视频编码。

## 功能特性

- **三栏布局**：左栏视频列表 | 中栏预览+波形+时间轴 | 右栏分类+字幕
- **视频编辑**：播放/暂停（Space）、切割（S）、删除切片（Del）、撤回（Ctrl+Z）、还原
- **音频波形**：实时波形显示，点击跳转，拖动播放头
- **字幕识别**：基于 RapidOCR 的 OCR，支持框选 ROI 区域，自动合并相似字幕
- **分类导出**：一键分类并导出切片，自动命名 `【分类名】字幕.mp4`
- **跳过切片**：一键跳过不需要的切片
- **字幕勾选**：字幕列表支持勾选，导出时仅使用选中的字幕组成文件名
- **拖动排序**：分类列表和字幕列表均支持拖动调整顺序
- **自动推进**：分类或跳过后自动前进到下一个切片，当前视频全部处理完自动切换到下一个未处理的视频
- **GPU 加速**：OCR 推理支持 DirectML，视频导出自动使用 NVENC（回退 libx264）
- **音量调节**：播放音量 dB 滑块（不影响导出）
- **配置记忆**：自动保存源目录、输出目录、音量、OCR 设置等，重启后自动恢复并扫描
- **数据持久化**：项目状态保存在 `clip_index.json`，支持增量扫描

## 环境要求

- [Node.js](https://nodejs.org/) >= 18
- [FFmpeg](https://ffmpeg.org/)（系统 PATH 中，或放入 `runtime/ffmpeg/`）
- [Python](https://www.python.org/) >= 3.10（OCR 功能需要）

### OCR 依赖（Python）

```bash
pip install rapidocr_onnxruntime onnxruntime-directml
```

> 也可使用 `onnxruntime`（CPU）或 `onnxruntime-gpu`（CUDA），OCR 会自动选择可用后端。

## 安装与启动

```bash
cd electron-app

# 安装依赖
npm install

# 开发模式（自动打开 DevTools）
npm run dev

# 生产模式
npm start
```

## 使用方法

1. 选择 **源目录**（包含视频文件）和 **输出目录**（分类结果保存位置），选择后自动扫描
2. 左栏选择视频，中栏预览播放
3. 使用 **切割/删除** 工具编辑切片
4. 框选 **ROI** → 点击 **字幕识别** → 自动 OCR 并合并字幕
5. 手动编辑字幕（双击）或添加/删除/拖动排序字幕，勾选需要的字幕
6. 右栏添加分类，点击分类按钮一键导出；点击 **跳过** 忽略不需要的切片

### 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Space` | 播放/暂停 |
| `S` | 在播放头位置切割 |
| `Delete` | 删除选中切片 |
| `Ctrl+Z` | 撤回 |
| `←` / `→` | 后退/前进 1s |
| `Shift+←` / `Shift+→` | 后退/前进 5s |
